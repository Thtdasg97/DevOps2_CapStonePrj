## 🗺️ Tổng quan kiến trúc CI/CD

┌────────────────────────────────────────────────────────────────┐
│  Dev machine                                                   │
│   git push → GitHub (branch: main)                            │

          ▼                                 ▼
┌─────────────────┐               ┌──────────────────┐

│  1. Checkout    │               │  1. Checkout      │
│  2. Build image │               │  2. Build image   │
│  3. Push → Hub  │               │  3. Push → Hub    │
│  4. SSH → VPS   │               │  4. Deploy on VPS │
│     pull + up   │               │                  │

         │ deploy                          │ deploy
         └──────────────┬──────────────────┘

          ┌─────────────────────────┐
